## 📌 Sobre o projeto

O **We Care** foi um projeto desenvolvido com o objetivo de colocar em prática os conhecimentos adquiridos em aula, principalmente na criação de páginas com:

- HTML
- CSS

Neste projeto, foquei na construção da estrutura da página, estilização dos elementos e organização visual da interface.

---

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3

---

## 🎯 Objetivo

O principal objetivo deste projeto foi treinar:

- estruturação de páginas com HTML
- estilização com CSS
- organização de layout
- posicionamento de elementos na tela
